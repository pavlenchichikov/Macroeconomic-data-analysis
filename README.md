# Macroeconomic Data Analyzer

Скрипт для анализа макроэкономических показателей стран на основе CSV-файлов. Обрабатывает предоставленные файлы и выводит сводный отчет в консоль.

## Установка и запуск

1. Клонируйте репозиторий и перейдите в папку проекта.
2. Создайте виртуальное окружение и установите зависимости:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Windows: venv\Scripts\activate
   pip install -r requirements.txt
3. Запустите скрипт, передав пути к файлам и название отчета:
   Например,
   Bash
   python main.py --files economic1.csv economic2.csv --report average-gdp
